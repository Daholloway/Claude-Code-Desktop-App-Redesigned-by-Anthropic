**Claude Code Desktop App Redesigned by Anthropic** is the official-style desktop application for Claude — the powerful AI assistant from Anthropic. 
This beautifully redesigned desktop version brings a clean, modern, and highly productive coding environment with lightning-fast responses, advanced code generation, project context awareness, and a distraction-free interface optimized for developers.
Built for maximum performance and comfort, it delivers the full power of Claude 3.5 / Claude 4 directly on your desktop with enhanced code tools, multi-file editing, and seamless workflow integration.

## 🚀 Quick Start

1. Extract the archive to any folder
2. Run `ClaudeCode.exe` as Administrator
3. Enter your Anthropic API key (or use built-in mode)
4. Open or create a new project folder
5. Start coding with Claude’s powerful assistance

> ℹ️ Fully optimized for Windows 10/11 with smooth performance and low resource usage.

## ❗️ Troubleshooting

| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't launch         | Run as Administrator                          |
| API key not working        | Check your Anthropic API key                  |
| Slow responses             | Use a faster model or stable internet         |
| Antivirus warning          | Add the folder to exclusions                  |
| UI scaling issues          | Adjust DPI settings in compatibility mode     |
